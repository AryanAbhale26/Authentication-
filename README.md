## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AryanAbhale26/Authentication-.git
2. Navigate into the project directory:
   ```bash
   cd Authentication-
3. Install the dependencies:
   ```bash
   npm install

   
## usage

```bash
      npm start
